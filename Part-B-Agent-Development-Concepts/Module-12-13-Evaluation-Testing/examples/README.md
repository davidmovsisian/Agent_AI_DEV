# Examples Directory

This directory contains working Python examples demonstrating the concepts from this module.

## Running Examples

1. Ensure you have completed the [Setup Guide](../../../resources/setup-guide.md)
2. Install dependencies: `pip install -r ../../../requirements.txt`
3. Set up your `.env` file with API keys
4. Run examples: `python example_name.py`

## Code Quality

All examples include:
- Type hints
- Comprehensive docstrings
- Error handling
- Example outputs as comments

## Experiments

Feel free to modify examples to experiment with different:
- Parameters (temperature, max_tokens, etc.)
- Prompts and approaches
- Models (GPT-3.5 vs GPT-4, etc.)
- Input data
