# Final Project Submission Guidelines

## Submission Requirements

### 1. Code Repository
**Location**: Public GitHub repository  
**Structure**:
```
/
├── README.md
├── requirements.txt
├── .env.example
├── src/
│   ├── agents/
│   ├── tools/
│   ├── workflows/
│   └── main.py
├── tests/
├── docs/
└── examples/
```

### 2. Documentation
- README with setup and usage
- Architecture document
- API documentation
- Design decisions

### 3. Demo
- 5-10 minute video
- Shows key features
- Explains architecture
- Demonstrates capabilities

### 4. Project Report
- 2-3 pages
- Problem statement
- Solution approach
- Results and evaluation
- Learnings and future work

## Grading Rubric

**Total: 100 points**

- Architecture & Design (25 pts)
- Implementation (40 pts)
- Testing & Evaluation (15 pts)
- Documentation (10 pts)
- Demo & Presentation (10 pts)
